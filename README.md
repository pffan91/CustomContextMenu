# Custom Context Menu

A UIKit-based iOS demo showing a fully custom context menu implementation. It intercepts `UIContextMenuInteraction` to present a styled modal menu with smart positioning, badges, haptic feedback, and accessibility support. Works with both `UITableView` cells and regular `UIView`s.

## Key Features

- Custom-styled modal menu presented via `UIContextMenuInteraction`
- Smart positioning relative to the source view
- Badge support on menu items
- Haptic feedback on interactions
- Full accessibility / VoiceOver support
- Works with `UITableView` cells and standalone `UIView`s
